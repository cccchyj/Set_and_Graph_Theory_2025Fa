# 鸽笼原理

### 5.1 在边长为 2 的正三角形中任意放置 5 个点，证明至少有两个点之间的距离不大于 1。

**证明**：将边长为 2 的正三角形的各边中点两两相连，可把大三角形分成 4 个全等的小正三角形，每个小三角形边长为 1。把 5 个点放入这 4 个小三角形中（点在分割线上时，按任一侧算），根据抽屉原理，至少有两个点落在同一个小三角形内。由于同一小正三角形内任意两点的距离不超过其最长边长 1，所以至少存在一对点之间的距离不大于 1。

### 5.2 设 $a_1, a_2, \dots, a_n$ 为整数，则存在 $k$ 和 $l$（$1 \leq k < l \leq n$），使得 $a_k + a_{k+1} + \dots + a_l$ 被 $n$ 整除。

**证明**：记**前缀和** $S_k = a_1 + a_2 + \cdots + a_k$（$k = 1, 2, \cdots, n$）。

若存在 $S_k \equiv 0 \pmod{n}$，则 $a_1 + \cdots + a_k$ 能被 $n$ 整除，结论成立。

否则，所有 $S_1, \dots, S_n$ 模 $n$ 的余数均不为 0，此时仅有 $n - 1$ 个非零余数，而有 $n$ 个前缀和。由抽屉原理，必有 $S_i \equiv S_j \pmod{n}$（$i < j$）。于是：
$$S_j - S_i = a_{i + 1} + \cdots + a_j \equiv 0 \pmod{n}$$
取 $k = i + 1$、$l = j$，即得所求。

### ✅5.3 证明：对于任意正整数 $N$，必存在 $N$ 的一个倍数，使得它仅由数字 0 和 7 组成。

**证明**：考虑数列 $7, 77, 777, \dots, \underbrace{77\cdots7}_{N + 1 \text{ 个 } 7}$，对这些数取模 $N$ 的余数（共 $N + 1$ 个数，模 $N$ 仅 $N$ 个不同余数）。

若某数模 $N$ 余 0，则该数是仅由 0 和 7 组成且为 $N$ 倍数的数。

若都不为 0，由抽屉原理，存在两个不同全 7 数模 $N$ 余数相同。用较长的全 7 数减较短的，差为若干个 7 后跟若干个 0（如 $7777000$），即差仅由 7 和 0 组成且能被 $N$ 整除，得到所需倍数。

### ✅5.4 把一个圆盘分为 36 段，将 1，2，…，36 这 36 个数字任意标在每一段上，使得每一段恰有一个数字。证明：必存在相继的三段，它们的数字之和不小于 56。

**证明**：设圆盘上顺时针数为 $x_1, x_2, \dots, x_{36}$（循环索引），考虑从位置 $i$ 开始的连续三段和 $T_i = x_i + x_{i + 1} + x_{i + 2}$（$i = 1, \dots, 36$）。每个 $x_j$ 在 36 个和中恰出现 3 次，故：
$$\sum_{i = 1}^{36} T_i = 3\sum_{j = 1}^{36} x_j = 3 \cdot \frac{36 \cdot 37}{2} = 1998$$

三段和的平均值为 $1998 \div 36 = 55.5$。若每个三段和都 $\leq 55$，则总和 $\leq 36 \times 55 = 1980 < 1998$，矛盾。故存在 $i$ 使 $T_i \geq 56$。

### ✅5.5 证明下述程序能在 $2n$ 个不同的数中选取最大的 $n$ 个数。
（1）将 $2n$ 个数中的 $n$ 个数排成递增次序，设它们为 $a_1, a_2, \dots, a_n$。
（2）将剩下的 $n$ 个数排成递减次序，设它们为 $b_1, b_2, \dots, b_n$。
（3）对一切 $i = 1, 2, \dots, n$，比较 $a_i$ 和 $b_i$，并且选取它们中较大的一个数。

**证明**：记每对的两个数为 $(a_i, b_i)$，对每个 $i$ 我们选 $\max(a_i, b_i)$，不被选的是 $\min(a_i, b_i)$。

**关键要证：对任意 $i, j$，有
$$\min(a_i, b_i) \leq \max(a_j, b_j)$$

分两种情况讨论：
- 若 $i \leq j$：  因为 $a$ 是升序排列，所以 $a_i \leq a_j$。
  又 $\min(a_i, b_i) \leq a_i \leq a_j \leq \max(a_j, b_j)$，故不等式成立。
- 若 $i > j$：  因为 $b$ 是降序排列，所以 $b_i \leq b_j$。
  又 $\min(a_i, b_i) \leq b_i \leq b_j \leq \max(a_j, b_j)$，故不等式成立。

由此，**任一对中的“小者”都不超过任一对中的“大者”**。所有 $n$ 个“小者”组成原 $2n$ 个数中较小的 $n$ 个，所有 $n$ 个“大者”组成原 $2n$ 个数中较大的 $n$ 个。而算法恰好选取了每对的“大者”，即原 $2n$ 个数中最大的 $n$ 个，故算法正确。

### ✅5.6 从 $i = 1, 2, \dots, 100$ 中任取 51 个数，证明：
（1）其中必有两个数，它们的差是 50。
（2）其中必有两个数，使得一个是另一个的倍数。

**证明**：设取的 51 个数为集合 $S$。

（1）将 $1$ 与 $51$、$2$ 与 $52$$…$$50$ 与 $100$ 分为 50 对，取 51 个数，由抽屉原理，至少有一对的两个数都在 $S$ 中，它们的差为 50。

（2）**将数表示为 $2^k \cdot m$（$m$ 为奇数，即“奇数部分”）**，1 - 100 中奇数部分仅 50 种（$1, 3, \dots, 99$）。51 个数按奇数部分分类，由抽屉原理，必有两数 $x = 2^r m$、$y = 2^s m$（$r < s$），则 $y = 2^{s - r}x$，即一个是另一个的倍数。

### ✅5.7 证明：在任意选取的 $n + 2$ 个整数中，一定存在两个整数，或者它们的差能被 $2n$ 整除，或者它们的和能被 $2n$ 整除。

**证明**：将整数按模 $2n$ 的余数分类：类 $0$（余 0）、类 $n$（余 $n$），及对 $r = 1, \dots, n - 1$，将余 $r$ 和余 $2n - r$ 合并为一类（因 $r + (2n - r) \equiv 0 \pmod{2n}$），共 $n + 1$ 类。把 $n + 2$ 个整数放入这些类，由抽屉原理，必有两数在同一类。若余数相同，差能被 $2n$ 整除；若余数互补，和能被 $2n$ 整除。

### 5.8 💡Erdős–Szekeres 引理

**命题：任取 $mn + 1$ 个不同实数的序列，必存在长至少 $m + 1$ 的递增子序列，或长至少 $n + 1$ 的递减子序列。**

**证明**：设序列为 $x_1, x_2, \dots, x_{mn + 1}$，对每个 $i$，**定义 $p_i$ 为以 $x_i$ 结尾的最长递增子序列长度**。

若存在 $p_i \geq m + 1$，则有长 $m + 1$ 的递增子序列。

否则 $1 \leq p_i \leq m$，共 $m$ 种取值。因有 $mn + 1$ 个 $p_i$，由抽屉原理，存在 $n + 1$ 个下标 $i_1 < i_2 < \cdots < i_{n + 1}$，使 $p_{i_1} = p_{i_2} = \cdots = p_{i_{n + 1}} = t$。

断言 $x_{i_1} > x_{i_2} > \cdots > x_{i_{n + 1}}$：若 $i_j < i_k$ 且 $x_{i_j} < x_{i_k}$，则以 $x_{i_j}$ 结尾的长 $t$ 递增子序列可接 $x_{i_k}$，得 $p_{i_k} = t + 1$，与 $p_{i_k} = t$ 矛盾。故存在长 $n + 1$ 的递减子序列。
